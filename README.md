# enterprise-risk-assessment-framework
Enterprise Risk Assessment Framework aligned with NIST SP 800-30 and ISO 27001, including risk register and scoring methodology
Risk scoring methodology (Likelihood x Impact)
- Control effectiveness evaluation
- Residual risk calculation

## Methodology
- Likelihood and Impact scored on a 1–5 scale
- Inherent Risk = Likelihood x Impact
- Residual Risk = Adjusted Likelihood x Impact
- Control effectiveness factored into adjusted likelihood

## Control Framework Alignment
- NIST SP 800-30
- ISO 27001

## Use Case
Applicable for financial services, legal, and regulated environments requiring structured risk governance.
